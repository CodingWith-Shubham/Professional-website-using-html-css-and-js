What is a list?

Lists are used to store multiple items in a single variable. Lists are one of 4 built-in data types in Python used to store collections of data, the other 3 are Tuple, String and Dictionary, all with different qualities and usage. Lists are created using square brackets. Example: thislist = ["apple", "banana", "cherry"]

How characters of Lists are accessed?

In order to access the list items refer to the index number. Use the index operator [ ] to access an item in a list. The index must be an integer. Nested lists are accessed using nested indexing.

Deleting or Updating a List

You can update single or multiple elements of lists by giving the slice on the left-hand side of the assignment operator, and you can add to elements in a list with the append method. You can use methods such as del, remove, sort, insert, pop, extend etc to update a list.

List Operations

Just like string and tuple we can do various operations in a List which include concatenation, repetition, membership testing, slicing, indexing. We can concatinate a list with '+' , repitition with '*', membership with 'in' and 'not in' operators.
